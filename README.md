# Mental Health Risk Prediction System

##  Problem Statement
This project aims to build an AI-powered system that predicts mental health risk levels based on workplace environment, personal factors, and lifestyle indicators. The system helps individuals, HR teams, and organizations take preventive actions.

---

## 📊 Dataset
- Source: Kaggle Mental Health in Tech Survey  
- Features include age, gender, work interference, remote work, company support, and more.

---

##  Features Used
- Age  
- Gender  
- Family History  
- Work Interference  
- Remote Work  
- Company Benefits  
- Care Options  

---

##  Machine Learning Models Used
- Logistic Regression  
- Random Forest  
- XGBoost  
- Support Vector Machine (SVM)  

### 📈 Model Performance
- Accuracy: > 80%  
- F1 Score: Used due to class imbalance  

---

##  Data Preprocessing
- Handled missing values  
- Encoded categorical features  
- Feature scaling / normalization  

---

##  Feature Engineering
- Work stress score  
- Work-life balance indicator  
- Remote vs office work  
- Company support index  

---

## 💻 Streamlit Application Features

### 🔹 Prediction System
- Inputs :
  - Age  
  - Gender  
  - Family History  
  - Work Interference  
  - Remote Work  
  - Company Benefits  
  - Care Options  

- Outputs:
  - Risk Level (Low / Medium / High)  
  - Risk Percentage  

---

### Recommendation Engine
- Provides suggestions based on risk level:
  - Improve work-life balance  
  - Seek professional help  
  - Engage in wellness programs  

---

### 📊 Visualization Dashboard
- Stress vs Company Support  
- Age vs Stress  
- Work Environment vs Stress  

---

### ⚖️ Bias Detection Analysis
- Gender-wise stress comparison  
- Age-wise stress comparison  
- Company size impact analysis  

---

### 🤖 AI Support Chatbot
- Detects user emotions (stress, sadness, anxiety)  
- Context-aware conversation  
- Provides supportive responses  

---

## 🎯 Results
The system successfully predicts mental health risk levels with good accuracy and provides actionable insights for users.

---

## 🚀 Conclusion
This project helps in early detection of mental health risks and supports better decision-making for individuals and organizations.
